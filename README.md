# avatar-wechat-mini
avatar项目的微信小程序

## Avatar项目微信小程序端
 1. 将工程下载到本地：git@github.com:wligang/avatar-wechat-mini.git
 2. 将您的小程序域名配置到[http://www.wlgdo.com]的域名下
 3. 导入到微信小程序开发工具中，因为我的后台接口是完全开放的，所以您可以直接启动运行该项目。


### Hido互联
这个工程已经发布完成了，在微信小程序搜索“Hido互联”或者扫描下面的小程序码,即可体验
