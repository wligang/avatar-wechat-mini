# avatar-wechat-mini
avatar项目的微信小程序

## Avatar项目微信小程序端
++ 将工程下载到本地：git@github.com:wligang/avatar-wechat-mini.git
++ 将您的小程序域名配置到![https://www.wlgdo.com](https://img-blog.csdnimg.cn/20190505115944855.jpg)的域名下
++ 导入到微信小程序开发工具中，因为我的后台接口是完全开放的，所以您可以直接启动运行该项目。
